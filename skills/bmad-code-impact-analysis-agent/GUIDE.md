# BMAD Code Impact Analysis Agent — Setup Guide

Code impact analysis for enterprise projects.

---

## Prerequisites

- Python 3.10+
- BMAD installed on your project

## Installation

Installed as part of the BMAD Code Audit module:

```bash
npx bmad-method install \
  --directory . \
  --modules bmm,bmb \
  --custom-source /path/to/bmad-code-audit/skills \
  --tools claude-code \
  --yes
```

After install: `.claude/skills/bmad-code-impact-analysis-agent/`
